Enter a search string to find messages whose author, subject, and body contain contain all of the given words.

The following operators are supported:

  • from:name - show only messages whose author contains the given name
  • subject:"some words" - show only messages whose subject contains the given words
  • body:"some words" - show only messages whose body contains the given words

Note that exact matches are not supported yet. For example, a search for "very serendipitous" in quotes will still return results where the words occur separately in the message, along with variations like "serendipitously"
